Array: Bunch of Data.
Array in Js is dynamic, that means we can edit it (add some data as much as we want), not like Java when we need to has the long of array and it's fix.
How to create it? Use [] for empty array
Use ["data1", "data2", "data3"] for array with value inside
How array works?
1. Start from 0
2. The data that added always come to the end or last of the queue
3. queue in array called index


How to edit an array?
1. arrayName.push : add
2. arrayName.length : get array length
3. arrayName[index]: get the position index of the data
4. arrayName[index] = value : change the data from that position
5. delete arrayName[index] : delete the data in that index position, it's only delete the data, the index still there, in case you gotta have some others data value in the future


in Js we can have array in array (multidimension array)
