Error Handling

Note:

For anticipate Error Handling we can use try & catch
Try will find if we have error or not
Catch will do something with that error


Actually we have another keyword to use in here, namely: finally
here in finally it will always execute after try and catch
different with catch, catch only will execute if the try have some error, but finally always execute even the try have error or not
so it will get execute the try, catch then finally for any cases
also we can straight use try and finally without the catch
