Prototype

Note:

We use a lot or almost everytime use it in Js: Prototype
Prototype relate to Inheritance
Everytime we make a constructor function we always automatically create the prototype of it,
let say we make a constructor function Person it will automatically create Person.prototype
When we make an object, automatically that object is inherit from the Constructor.prototype
To access it we use __proto__


What is this for:
We store some method in prototype to make it efficient and we dont store it in instance object to save the memory (more efficient)
then let say if we have same function's name in instance object and the prototype, when we call that function the one that get call is the one in instance object, so it's checking one by one from the parent to child
