Logic in Non Boolean

Note:

In Js we can logic operator for Non Boolean.
We can use && and ||

OR:
we use this a lot.
1. Read from Left to Right
2. Take first truthy value
3. If there is no truthy value, the last value will be taken.


AND:
1. Read from Left to Right
2. Take first falsy value
3. If there is no falsy value, the last value will be taken.
