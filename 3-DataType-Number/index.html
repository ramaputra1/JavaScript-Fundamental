Number Data Type

Note:


1. Different with other programming language, JavaScript only support one type of number namely: number, where number can be whole or decimal. So, one for all.
How?
Just as simply as type the number.
a. Whole: 100
b. Decimal: 100.01


2. Number Annotation.
We can change the base of number in Js.
Hexadecimal, start with 0x
Binary, start with 0b
Octal, start with 0o
