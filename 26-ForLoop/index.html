Note:

In for loop, if the consition always satisfied, it will always run.
syntax: for(init statement;condition;post statement){your block}
1. Your block: it will always iterate until the condition not satisfied.
2. Init: will execute once in first time.
3. condition: check the condition, if true execute until it false.
4. post statement: will execute every time in the end of looping (init -> condition -> looping -> post statement -> condition -> looping -> post -> condition)
Init statement, condition and post statement is not mandatory, if the condition is empty that means it's true. (will never stop)
