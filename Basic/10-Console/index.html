Console

Note:

What is Console? Console it means to use the feature of logging,
What is that?
It is when you want to tell some information as a programmer such as for debugging or even some text,
but we do not want to show it in the web page (Because that will distract or annoy the user), so we use the console feature in web browser to show those kind of information.

We can simply click right at mouse and choose inspect then check the console.

All console that we have: 1. console.info() : tell some information
2. console.warn() : tell some warning information
3. console.error() : tell some error information
4. console.table() : tell information in table
