This Keyword

Note:

We use This to refers to who it belongs,
every this can have diffrent references, depends on the location

1. Object Method: This will refer to object that has the function
2. Global Scope: Refer to global object (window ussually)
3. Strict Mode: Refer to undefined
4. Event: Refer to element whos receive the event (in DOM)
