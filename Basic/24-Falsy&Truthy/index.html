Falsy & Truthy

Note:

In javascript, not only boolean false that considered as false
Some of data is also considered as false or we grouping that as falsy.

Falsy Group:
1. false (boolean)
2. 0,-0
3. "" , '' , `` if the string is empty
4. null
5. undefined
6. NaN


Besides of them, the rest is Truthy or considered as true
