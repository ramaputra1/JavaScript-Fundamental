Optional Chaining

Note:

Optional chaining is an operator to help us avoiding error because of nullish data. (Happens if we want to access a property that actually is nullish from an object)
If we want to access some property from nullish data, it will throw an error.
So to prevent that we use optional chaining

We will use ? in each property to check if this property is nullish, the rest of property will be nullish as well.
