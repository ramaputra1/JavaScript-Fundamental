Strict Mode

Note:

Since Ecma 5, we can use strict mode in Js. The normal one that we ussually use is sloppy mode
What changed:
1. Changes the silent error to throw error (shown)
2. Fix some engine for optimazion
3. Avoid some intructions that we do not use anymore in EcmaScript (more)
How to activatate the strict mode:
add "use strict" in the first line of our js code or before our function
