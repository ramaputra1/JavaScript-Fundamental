Variable

Note:


Why variable?
To storing data, so if we need that data we do not need to write them one by one, just call it.


Javascript is Dynamic language, that means when you want to declare a variable you do not need to tell the program what kind of data type is this(like in Java), in Js we can change the data type in variable cause it did not require to declare the data type first.



First keyword that we can use is var.
In Js like I said earlier, we can change the data in variable just using =

How to call: Just type the name of variable combine with the writeln or console or other.


CHANGES:
After 2015, since ECMAScript2015, we do not use var anymore, change it into let & const


Difference LET & CONST
let: we can change the value of the variable later.
const: not allowed to change the value of variable. (Ussualy for data like the name of app or something that cannot change later)
(constant)


IMPORTANT:
If you force to change your data value in const variable it will throw an error in console,
and if it's throw an error, the rest of your code after the error will never be executed, so carefull.
