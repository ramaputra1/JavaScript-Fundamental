Getter & Setter

Note:

Getter & Setter is another approach to make a function in object's property
So, in object we can make Getter & Setter, also it can do as much as what function can do

1. Getter: Take a data from object's property (No Parameter)
2. Setter: Change data from object's property


When we use get in object, it will count as property
