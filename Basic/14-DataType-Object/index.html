in another program (PHP), we can change the index from number to string (associative array), but in Js we can't change that, it's always default a number (can not use associative array),

so, if we want to use a string for the index, we use object.
one note: we will change the term index to attributes or properties

then, in array we use [] but in object we gonna use {}

some operators in object are the same like in array, the different is only that the index we will change with the properties.

1. arrayName[index] = value : change / add the data from that position
2. delete arrayName[index] : delete the data in that property position
