Rest Parameter & Spread Syntax

Note:

Rest Parameter is a feature where we can send as many as data to one parameter. and automatically will convert to Array
But there is terms to use Rest Parameter

Terms:
1. We can only use one in a function
2. It always need to be put in the last, for example if we have 2 parameter, our Rest Parameter need to be in the last

In other programming language they call it Variable Argument

How to use it:
add "..." before our parameter's name to be Rest Parameter

Then, in some cases sometimes we already have an array.
so in Js, we dont have feature to convert array using Rest Parameter
but we use Spread Syntax to convert array to Rest Parameter
we still using "..."
