typeOf Operator

Note:

It's an operator to check what is the data type of value or variable
Why need this?
JavaScript is a dynamic language, that means that we can have multiple data types in one variable array or object, that is why sometimes we use typeOf to check what kind of data type of that value in some variable.

The output of typeOf would be string that data type.

List of data we can check:
1. Undefined
3. Boolean
4. Number
5. BitInt
6. String
7. Symbol
8. Function
9. Except of all of these things it would return: Object
